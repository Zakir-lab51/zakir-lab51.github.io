PYQ 01

Which protocol provides secure transmission?

Answer: HTTPS

Explanation: HTTPS encrypts data between the web server and browser.

PYQ 02

What is the full form of CPU?

Answer: Central Processing Unit

Explanation: CPU is the main processing unit of the computer.

PYQ 03

Which view arranges slides as thumbnails?

Answer: Slide Sorter View

Explanation: This view helps arrange, copy, and delete slides easily.

PYQ 04

What is a computer virus?

Answer: Malicious self-replicating code

Explanation: It can spread by attaching to programs or files.

PYQ 05

Which software is used for spreadsheets?

Answer: MS Excel

Explanation: It manages tabular data using rows, columns, and formulas.

PYQ 06

What does DNS do?

Answer: Converts domain names into IP addresses

Explanation: It helps browsers find web servers.
